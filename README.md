## 🌟 About Samhit

**Samhit** (संहित) is a Sanskrit word meaning **"union"** or **"coming together."** In the context of mental health, it represents the union of mind, body, and spirit - the harmonious integration of our mental well-being with our overall health. Our platform embodies this philosophy by bringing together awareness, support, and healing.

## 🎯 The Problem We Solve

Many people struggle with mental health challenges without realizing they need help. The stigma around mental health, lack of awareness, and subtle onset of symptoms often prevent individuals from recognizing their struggles or seeking support. 

> 🚨 **Did you know?** 1 in 4 people worldwide are affected by mental health disorders, yet many go undiagnosed and untreated.

Samhit bridges this critical gap by providing:

| 🎯 **Early Detection** | 📚 **Awareness** | 🤝 **Support** | 🌍 **Community** |
|:---:|:---:|:---:|:---:|
| Identify potential concerns before they escalate | Educate about symptoms and conditions | Connect with resources and professional help | Safe space for shared experiences |

## 🔮 Vision

> *To create a world where mental health is prioritized, understood, and supported - where no one suffers in silence because they don't recognize their struggles or know where to turn for help.*

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- ![Node.js](https://img.shields.io/badge/Node.js-v14+-green.svg) Node.js (version 14 or higher)
- ![npm](https://img.shields.io/badge/npm-latest-red.svg) npm or yarn
- ![Browser](https://img.shields.io/badge/Browser-Modern-blue.svg) Modern web browser

### 📦 Installation

Follow these steps to get Samhit running locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aditya1286/Samhit.git
   cd Samhit
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start the development server**
   ```bash
   npm start
   ```

5. **Open your browser**
   ```
   Navigate to http://localhost:3000
   ```
